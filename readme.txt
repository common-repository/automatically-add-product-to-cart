== Description ==

The Auto Add to Cart for WooCommerce plugin offers a powerful and user-friendly solution to enhance your e-commerce store by automatically adding products to the cart based on your configuration. This plugin simplifies the shopping experience for your customers by automatically including selected products in their cart as soon as they visit your site.

Features of the plugin include:

 * Provide complete solution for Automatically add product to cart in our E-commerce site.
 * it include Option to configure all product which you want to allow as automatically into cart when customer visit site.
 * Simple-configuration - just install,setup and enjoy.
 
 == Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `automatically-add-product-to-cart.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the menu name product.
4. Edit the product which you want to add Automatically to the cart page.
5. Now we can see a new option called "Add Product To Cart" in the Product Data tab and Enable the option to automatically add the product to the cart on-site visit

== Frequently Asked Questions ==

= How do I activate the plugin in WordPress? =

After uploading the plugin, go to your WordPress dashboard, then navigate to 'Plugins' > 'Installed Plugins'. click on 'Add New Plugin' Search 'WooCommerce' in the list and click on 'Dowanload' and 'Activate' to enable the plugin.

= How do I access the plugin settings? =

Go to the dashboard, click on "Products," and select the product you want to automatically add to the cart. In the "Product Data" section, click on "Add Product to Cart" and enable the option to automatically add the product to the cart on-site visits.
